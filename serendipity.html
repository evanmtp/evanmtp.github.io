SERENDIPITY

Hadi Jamali
Project link
Video by Hadi Jamali

Serendipity is an interactive audiovisual installation that provides an experience of getting access to multilayered content through viewers’ participation.

The detailed collage animation illustrates tales appropriated from Persian miniatures. By obscuring the viewer’s access to these narratives, the installation contemplates how having multiple perspectives can help us transcend our limited understanding as individuals and remain open to continuous discovery by viewing a collaborative effort.

Role

Using Kinects and OpenCV, I developed an interactive system that tracks the position of multiple visitors and draws vignettes revealing the underlying animation. The initial version of the project worked with a single Kinect. In 2023, I revised the project to allow input from two or more Kinects, making it possible to adapt the project to larger exhibition spaces.

Tools

Max, Kinect, OpenCV, GLSL
